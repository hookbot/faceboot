CPPS.me Commands

This is a list of all the commands we have available for our players. These can range from adding clothing items, to changing your nameglow, to checking a server's response time.

Player Commands

  • !AI <ID> - Replace <ID> with item ID you want to add to your inventory.
    • !AI 413 - Adds beta hat to your user.
  • !JR <room_id or room_name > - Replace <room_id | room_name > with the ID of the room or the name of the room you would like to join.
  • !AF <ID> - Replace <ID> with the ID of the furniture you want to add.
  • !AS <ID> - Replace <ID> with the ID of the stamp you want to add.
  • !PIN *ID* - Allows you to use any item as pin, replace *ID* with the item's ID.
  • !UP [{c|h|f|n|b|a|e|l|p}] <ItemID> - [the placement code is optional] Wear ItemID
  • !UP *INITIALS* - Put the initials of a 'mascot' - eg: RH (Rockhopper), AA (Aunt Arctic), etc, or try combinations such as !UP Rockhopper, as some of them are coded for ease of use!
  • !ROOMID - Shows the ID of your current room.
  • !AC <coins> - Replace <coins> with amount of coins you want to add.
    • !AC 5000 - Adds 5000 coins to inventory.
  • !CLONE <UserName> - Copies the outfit of another user
    • Usage: !CLONE Help - Copies the outfit of Help.
  • !NAME <UserID> - Fetches the username of the id it is associated with.
    • Usage: !Name 27600 - Returns the name of the id which is Help.
  • !ID [<UserName>] - Fetches the id associated with the username.
    • Usage: !ID Help - Returns 27600 which is the ID of Help.
  • PAID FEATURES - Features bought in manager or won in giveways and contests
    • !NG <nc> <ng> - nc changes name color, ng changes name glow
      • Usage: !NG FF0000 0000FF - Sets text color to red and glow to blue
    • !BC <tc> <bc>
      • Usage: !BC FF0000 0000FF - Sets text color to red and bubble color to blue
    • !WOW - Toggles walk on walls
      • Usage: !WOW - Turns on or off powers
    • !SPEED <Speed> - sets penguin speed ingame
      • Usage: !SPEED 10 - Sets speed to 10
  • !COLOR <HexCode> - Changes penguin color to the specified hexcode
    • !COLOR 0000FF - Changes penguin color to blue
  • !<COLOR> - Changes penguin color to the specified color
    • !PINK
    • !GOLD
    • !GRAY
    • !GREY
    • !GREEN
    • !RED
    • !WHITE
    • !BLACK
  • !HELP - Sends an alert to a Moderator ingame that you need help.
  • !IGLOO <IglooID> - Changes your igloo to the igloo id.
  • !FLOOR <FloorID> - Changes floor Id of igloo
  • !MUSIC <MusicID> - Changes music id playing in glood

Server Commands

These commands are for advanced users who are curious about server stats and information.

  • !PING - This command was made to check if you're still connected to the server.
  • !PONG - This command was made to check if you're still connected to the server.
  • !UPTIME - Shows the server's uptime.
  • !COUNT - Shows the amount of online users.
